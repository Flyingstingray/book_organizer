Installation Instructions

This apk is for a mobile application. It can be ran from a physical mobile device, or from an android emulator.
Now that you have accessed the html page to download the application, please click either link above. The release link will show you the release version, relase notes, and what is part of the release. The release link has a link to download the apk. The apk link brings you directly to the page to download the apk.
If you are downloading onto an android studio emulator, you can download the apk and then drag the file and drop it into the active emulator. If you are using a physical android device, you can just click the download button. Your phone may prompt you to allow downloads from unkown sources, then confirm and install.
When downloading from a physical device you may have to go into settings and "Allow Downloads from this Source" before the device lets you download the app.
